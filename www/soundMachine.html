Sound Machine

Deciding to make this app was easy, deciding how to build it was difficult. The only thing that I wanted was that it would be built to be cross-platform.

At this point I really only knew Java and I knew that it was cross-platform enough. And so I set off to do my project in Java and ended up completely scrapping the project when trying to get a cool UI to come together.

Discord, the voice chat application, ended up being a huge inspiration. Its UI looking clean and clear. Hoping to build something that was as usable as discord, I looked into how they were built and found out about Electron.

Electron is a framework that runs node.js code in what appears to be a native application but is actually what is basically a Chrome window.

I ended up having to learn Javascript/CSS/HTML, the all of it, to make this possible. Node.js's npm made much of this application. From running the music through howler.js to doing the networking with express/socket.io, the featureless became featurefull with all of the easily accessible and usable packages.
